# Archive Fresh Live Gemini Suite Runs

Labels: `evaluation`, `reporting`

## Summary

Refresh the checked-in live Gemini evidence with one current `gemini-core` run and one current `contributor-workflows` run. The goal is not to make the numbers look better; the goal is to keep the public evidence current and inspectable.

## Acceptance Criteria

- Run the live `gemini-core` suite and archive the latest report/results pair.
- Run the live `contributor-workflows` suite and archive the latest report/results pair.
- Refresh the README live-evidence table and reproducible-run metadata table together.
- Keep per-task outcome tables in sync with the archived reports.
