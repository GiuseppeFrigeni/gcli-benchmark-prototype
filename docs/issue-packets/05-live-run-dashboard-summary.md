# Live Run Dashboard Summary

Labels: `reporting`, `documentation`, `good first task`

## Summary

Generate a compact machine-readable summary for archived live Gemini runs so the README evidence tables and case-study links can be refreshed without hand-editing multiple sections.

## Acceptance Criteria

- Emit one summary file per archived live-run directory.
- Include run metadata, suite/task counts, pass/fail breakdown, and artifact links.
- Keep the format simple enough that README refreshes can be scripted later.
- Document how the summary is refreshed.
