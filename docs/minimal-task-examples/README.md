# Minimal Task Examples

These examples are intentionally small, copyable starting points for each task kind.

- `workspace-edit/`: smallest repo-backed patch task
- `prompt-output/`: strict-output task scored from stdout
- `tool-use/`: investigation task scored from stdout plus normalized tool activity

Each example references [`../task.schema.json`](../task.schema.json) and is designed to be tightened before promotion into `tasks/`.
